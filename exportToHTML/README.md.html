2    # Parsing​ ​Metacritic​ ​Test / REST API
3    # Task:
4    
5    Write an application in which can do the following:
6    
7    1. Parse the HTML for the "Top Playstation 4 Games (By Metascore)" on Metacritic's PS4 page: `http://www.metacritic.com/game/playstation-4`.
8    
9    2. Expose a REST API for retrieving top PS4 games. There should be 2 exposed methods:
10   
11   ## GET​​ ​​/games
12   An​ ​HTTP​ `​GET`​ ​request​ ​to​ `​​/games`​​ ​returns​ ​all​ ​top​ ​PS4​ ​games​ ​on​ ​the​ ​Metacritic page​ ​as​ ​an​ ​array​ ​of​ ​JSON​ ​objects.​ ​For​ ​example:
13   
14   
15    ```json
16   [
17     {
18       "title": "Persona​ ​5",
19       "score": 94
20     }, 
21     {
22       "title": "Horizon:​ ​Zero​ ​Dawn",
23       "score": 89
24      }
25   ]
26   ```
27   
28   ## GET​​ ​​/games/<title> 
29    An​ ​HTTP​ ​GET​ ​request​ ​to​ ​​`/games/​<title>`​​ ​returns​ ​JSON​ ​for​ ​a​ ​specific​ ​game that​ ​matches​ ​the​ ​corresponding​ ​game​ ​title.​ ​
30    For​ ​example,​ ​an​ ​HTTP​ ​GET​ ​to `/games/Nioh​​` ​should​ ​return​ ​an​ ​individual​ ​JSON​ ​object​ ​for​ ​Nioh:
31    
32   ```json
33   {
34   ​ ​​ ​​ ​​"title":​​"Nioh", 
35   ​​ ​​ ​​ ​​"score":​ ​88
36   }
37   ```
38
